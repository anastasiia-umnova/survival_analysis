# Анализ выживаемости

# Описание проекта
В проекте использовано машинное обучение для создания модели, которая предскажет, какие пассажиры выжили после крушения.

# Входные данные
В качестве входных данных csv-файлы:
* train.csv обучающий набор
* test.csv тестовый набор.

# Анализ данных
* Исследовательский анализ данных.
* Визуализация распределения выживших.
* Обучение модели с использованием логистической регрессии, случайного леса, градиентного бустинга, метода опорных векторов.
* Сравнение точности моделей.

#  Выводы
SVM показала наилучшие результаты с точностью 82.1%, что указывает на ее эффективность в данной задаче классификации выживаемости пассажиров Титаника.

# Библиотеки и технологии
scikit-learn, pandas, seaborn, matplotlib.
